# AI Contract Reviewer

## Overview

AI Contract Reviewer is an intelligent document analysis application designed to simplify the review of legal and business contracts. The system processes uploaded PDF contracts, analyzes their content, and extracts key information to help users quickly understand important clauses and contractual details.

The project demonstrates the application of Natural Language Processing (NLP) techniques in legal document analysis through an easy-to-use web interface.

---

## Features

* Upload and analyze PDF contract documents
* Automated contract content processing
* Extraction of key contractual information
* User-friendly web interface built with Flask
* Fast and efficient document review workflow
* Structured presentation of analysis results

## How It Works

1. User uploads a contract in PDF format.
2. The application extracts text from the document.
3. NLP-based processing analyzes the contract content.
4. Important information and clauses are identified.
5. Results are displayed through a clean web interface.

## Installation

### Clone the Repository

```bash
git clone https://github.com/Peddireddy-Divya/AI-Contract-Reviewer.git
```

### Navigate to the Project Directory

```bash
cd AI-Contract-Reviewer
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

### Open in Browser

```text
http://127.0.0.1:5000
```
