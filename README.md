# AI Contract Reviewer

## Overview

AI Contract Reviewer is a web-based application that analyzes legal contracts and extracts important information from uploaded documents. The system helps users quickly identify key clauses and gain insights from contracts.

## Features

* Upload PDF contracts
* Automated contract analysis
* Extraction of key clauses and information
* User-friendly web interface
* Fast document processing

## Technologies Used

* Python
* Flask
* HTML
* NLP (Natural Language Processing)

## Project Structure

AI_Contract_Reviewer/

├── app.py

├── requirements.txt

├── templates/

│ ├── index.html

│ └── result.html

└── uploads/

## Installation

1. Clone the repository:
   git clone https://github.com/Peddireddy-Divya/AI-Contract-Reviewer.git

2. Install dependencies:
   pip install -r requirements.txt

3. Run the application:
   python app.py

## Usage

1. Start the Flask application.
2. Open the local web address in your browser.
3. Upload a contract PDF.
4. View the extracted contract analysis results.
